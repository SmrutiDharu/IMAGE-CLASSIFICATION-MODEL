## DESCRITPION OF TASK: IMAGE CLASSIFICATION MODEL 
##PROJECT OVERVIEW:

This project focuses on building an image classification system using a Convolutional Neural Network (CNN). The goal is to design, train, and evaluate a deep learning model that can automatically classify images into predefined categories. The project uses the CIFAR-10 dataset, which is a well-known benchmark dataset in computer vision and deep learning research.

##DATASET DESCRIPTION:

The CIFAR-10 dataset consists of 60,000 color images of size 32×32 pixels, divided into 10 different classes:

Airplane
Automobile
Bird
Cat
Deer
Dog
Frog
Horse
Ship
Truck

The dataset contains:
50,000 training images
10,000 testing images

Due to the small resolution (32×32), images may appear pixelated or blurry when visualized. This is a normal characteristic of the dataset and does not affect model performance.

##TRAINING AND EVALUATION:
The model is trained for 10 epochs using the training dataset, with validation performed on the test dataset. During training, both training accuracy and validation accuracy are monitored.

After training, the model is evaluated on the test dataset to measure its generalization performance. The final results show:

Training Accuracy: Approximately 80–85%
Validation/Test Accuracy: Approximately 70%
